# FEATURE-07: 출고 처리

## 기능 목적

CONFIRMED 상태의 주문을 출고 처리하여 RELEASE 상태로 전환한다.  
출고 완료 시 해당 주문 수량만큼 재고에서 차감된다.

---

## 사용자 시나리오

1. 사용자가 출고 처리 메뉴에 진입한다.
2. CONFIRMED 상태의 주문 목록이 표시된다.
3. 사용자가 출고할 주문을 선택한다.
4. 출고 처리가 완료되고 주문이 RELEASE 상태로 전환된다.
5. 주문번호, 출고 수량, 처리 일시가 출력된다.
6. 메인 메뉴로 돌아온다.

---

## 입력

- 출고할 주문 선택

## 출력

- CONFIRMED 주문 목록 (주문번호, 고객명, 시료명, 수량)
- 출고 완료 메시지: 주문번호, 출고 수량, 처리 일시

---

## 예외 처리 기준

- CONFIRMED 주문 없을 시: 빈 목록 안내 메시지 출력
- 유효하지 않은 주문 선택 시: 오류 메시지 출력 후 재입력 유도
