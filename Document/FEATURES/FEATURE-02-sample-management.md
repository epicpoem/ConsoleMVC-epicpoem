# FEATURE-02: 시료 관리

## 기능 목적

시스템에서 취급하는 시료를 등록하고 조회·검색할 수 있도록 한다.  
등록된 시료만 주문 대상이 될 수 있다.

---

## 사용자 시나리오

### 시나리오 A: 시료 등록
1. 사용자가 시료 관리 메뉴에 진입한다.
2. 시료 등록을 선택한다.
3. 시료 ID, 이름, 평균 생산시간, 수율을 순서대로 입력한다.
4. 시스템이 등록 성공 메시지를 출력한다.
5. 시료 관리 메뉴로 돌아온다.

### 시나리오 B: 시료 조회
1. 사용자가 시료 조회를 선택한다.
2. 등록된 전체 시료 목록과 현재 재고 수량이 표시된다.

### 시나리오 C: 시료 검색
1. 사용자가 시료 검색을 선택한다.
2. 검색 기준(ID / 이름 / 수율 등)을 선택한다.
3. 검색어를 입력한다.
4. 조건에 맞는 시료 목록이 표시된다.

---

## 입력

| 항목 | 설명 |
|------|------|
| 시료 ID | 고유 식별자 (예: S-001) |
| 이름 | 시료 명칭 (예: 실리콘 웨이퍼-8인치) |
| 평균 생산시간 | 1개당 생산 평균 시간 (단위: min/ea) |
| 수율 | 정상 시료 비율 (예: 0.92) |

## 출력

- 시료 목록: ID, 이름, 평균 생산시간, 수율, 현재 재고
- 등록 성공/실패 메시지

---

## 예외 처리 기준

- 중복 ID 입력 시: 오류 메시지 출력 후 재입력 유도
- 수율 범위 초과(0 미만 또는 1 초과) 시: 오류 메시지 출력
- 등록된 시료가 없을 때 조회/검색 시: 빈 목록 안내 메시지 출력
- 검색 결과 없을 시: 결과 없음 메시지 출력
