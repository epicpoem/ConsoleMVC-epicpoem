# FEATURE-06: 생산라인 조회

## 기능 목적

현재 생산 중인 시료와 생산 대기 중인 주문 목록을 조회하여 생산 현황을 파악한다.

---

## 사용자 시나리오

1. 사용자가 생산라인 조회 메뉴에 진입한다.
2. 현재 생산 중인 시료 정보가 표시된다.
3. 생산 대기 중인 주문 목록이 FIFO 순서로 표시된다.
4. 메인 메뉴로 돌아온다.

---

## 입력

- 없음 (진입 시 자동 조회)

## 출력

### 현재 생산 중
- 주문번호, 시료명, 주문량, 재고, 부족분, 실생산량, 진행률, 완료 예정 시간

### 대기 큐 목록 (FIFO 순)
- 순서, 주문번호, 시료명, 주문량, 부족분, 실생산량, 예상 완료 시간

---

## 예외 처리 기준

- 생산 중인 시료 없을 시: 안내 메시지 출력
- 대기 주문 없을 시: 대기 없음 안내 메시지 출력
