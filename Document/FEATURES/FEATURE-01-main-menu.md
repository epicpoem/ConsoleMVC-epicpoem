# FEATURE-01: 메인 메뉴

## 기능 목적

시스템 진입 후 사용자가 원하는 기능으로 이동하고 돌아올 수 있는 중앙 허브 화면을 제공한다.

---

## 사용자 시나리오

1. 사용자가 시스템을 실행한다.
2. 메인 메뉴 화면이 표시된다.
3. 사용자가 메뉴 번호를 입력한다.
4. 해당 하위 메뉴로 이동한다.
5. 하위 메뉴에서 작업을 마치면 메인 메뉴로 돌아온다.
6. 사용자가 종료 번호(0)를 입력하면 시스템이 종료된다.

---

## 입력

- 메뉴 번호 (숫자 1자리)

## 출력

- 메뉴 항목 목록:
  - [1] 시료 관리
  - [2] 시료 주문
  - [3] 주문 승인/거절
  - [4] 모니터링
  - [5] 생산라인 조회
  - [6] 출고 처리
  - [0] 종료

---

## 예외 처리 기준

- 유효하지 않은 번호 입력 시: 오류 메시지 출력 후 재입력 유도
- 빈 입력 시: 재입력 유도
